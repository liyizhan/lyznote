## Java基本类型
- ### 整型
    - byte
    - short
    - int
    - long
- ### 浮点类型
    - float
    - double
- ### 字符类型 char
- ### 真值类型 boolean

## 各类型取值范围

类型 | 存储需求 |取值范围
---|---|---
byte | 1字节 |-2^7^ ~ 2^7^-1
short | 2字节|-2^15^ ~ 2^15^-1
int | 4字节 |-2^31^ ~ 2^31^-1
long| 8字节 |-2^63^ ~ 2^63^-1
float|4字节|约±3.40282347E+38F（有效位数6-7位）
double|8字节|约±1.79769313486231570E+308(有效位数约15位)
char| 2字节|-2^15^ ~ 2^15^-1
boolean||

## 一些要注意的点
- char类型不建议使用
- boolean类型大小没有定义
